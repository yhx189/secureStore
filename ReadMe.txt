### Secure Store ###

================================================================================
DESCRIPTION:
An example of how to use the Secure Store file system, which behaves like the default file system except that all data stored on the device is encrypted, along with directory and file names, and paths must always be absolute, i.e., beginning with a “/” character.

For more help of how to build and install this application, please see BUILD_NOTES.
